Lista de definición

HTML
HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y diseñar páginas web. Se utiliza para estructurar el contenido en la web.
CSS
CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para describir la presentación de un documento escrito en HTML. Permite aplicar estilos, como colores y fuentes, a las páginas web.
JavaScript
JavaScript es un lenguaje de programación que permite implementar funcionalidades complejas en las páginas web. Se utiliza para crear interactividad y controlar multimedia, animaciones, y más.
